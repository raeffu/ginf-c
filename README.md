### Project folder

Simulate a CPU in C.

So far only the alu (arithmetic logic unit) works. To play around, execute:
```
> cd project/
> make alu-main
> ./alu-main
```

To test alu, execute
```
> project/test.sh
```

### Strings folder

Functions to learn how to cope with strings in C. To test, execute:
```
> cd strings/
> make
> ./strings
```
